## MVP vs Full Scope

### MVP (Minimum for Demo)

- [x] Architecture (Phase 1)
- [ ] Onboarding + Business Profile (Phase 2)
- [ ] Full agent pipeline + strategy page (Phase 3)
- [ ] Dashboard with scores (Phase 4 — simplified)
- [ ] Copilot with quick-ask (Phase 5 — no conversation history)
- [ ] Deployed (Phase 6)

### Full Scope (If Time Permits)

- [ ] WebSocket streaming for Copilot
- [ ] CSV KPI import
- [ ] Campaign generation sub-workflow
- [ ] Strategy history/versioning UI
- [ ] Feedback loop auto re-analysis

## Definition of Done (Per Feature)

1. API endpoint implemented with Pydantic validation
2. Frontend component connected to API
3. Agent output validated against schema
4. Data persisted to PostgreSQL
5. Basic error handling in place
6. Manually tested in dev environment
